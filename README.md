## 🎯 왜 Rust인가?

처음에는 평소에 궁금했던 Ruby로 시도하려고 했습니다. 

하지만 실제로 하룻동안 해보니 Python을 이미 아는 입장에서 Ruby는 너무 익숙했고, **진짜 도전**이 되지 못할 것 같았습니다.

우테코에서 말하는 진짜 도전을 해보고 싶었습니다. 그래서 선택한 것이 **Rust**입니다.

## 목표

1. Rust가 최근 각광받고 있는 이유를 체감하기
2. Rust가 JAVA와 무엇이 다른지를 비교하기
3. Rust 문법 익히기
4. 최종적으로, Rust를 이용해서 아주 작은 memoryDB 만들기
